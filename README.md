## 📖 Project Overview

The **Secure File Transfer Monitoring System (SFTMS)** is a cybersecurity application designed to monitor file system activities in real time. It detects file creation, modification, deletion, and movement, verifies file integrity using SHA-256 hashing, identifies sensitive and unauthorized access, records security events into a SQLite database, and provides an interactive web dashboard for monitoring, analytics, reporting, and audit logging.

The project demonstrates practical concepts used in:

* File Integrity Monitoring (FIM)
* Host-Based Intrusion Detection Systems (HIDS)
* Security Operations Centers (SOC)
* Digital Forensics
* Security Auditing
